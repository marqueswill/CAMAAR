    class TemplatesController

    A classe TemplatesController gerencia ações relacionadas ao gerenciamento de templates. Essa classe inclui a criação, edição, atualização, visualização e deleção de templates, assim como possui métodos de verificação de ações específicas de commit e definição de variáveis de instância a serem usadas nas views.

def update
  if !questions.present?
    flash[:alert] = "O template precisa conter pelo menos uma pergunta"
    return redirect_to edit_template_path(template, template: template_params)
  end

  if params[:template][:name].empty?
    flash[:alert] = "Template precisa de um nome"
    return redirect_to edit_template_path(template, template: template_params)
  end

  if template.update(template_params.merge(draft: false))
    redirect_to templates_path, success: "Template atualizado com sucesso!"
  end
end
