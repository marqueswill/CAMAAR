    class AdminsController

    import() click to toggle source

    Método que recebe uma requisição para importar algum dado a partir de um json. Caso haja sucesso ou erro, serão printadas as respectivas mensagens na página do administrador. Há o uso de uma instância da classe Import, responsável por executar os métodos de importação.

    # File app/controllers/admins_controller.rb, line 26
def import
  @errors = []
  json = JSON.parse(File.read(params[:admin_import][:file].tempfile.path))
  symbol, msg = Import.new.import_data(params[:select_data], json, current_admin.email)
  flash[symbol.to_sym] = msg
  redirect_to '/admins/import'
end
